# Project 2 – Menu Driven Python Program

## 📌 Project Overview
This project is a **menu-driven Python application** developed using **Jupyter Notebook (.ipynb)**.  
The program allows users to perform multiple operations through a simple interactive menu.

The project demonstrates:
- Python loops
- Conditional statements
- User input handling
- Pattern generation
- Number analysis
- Menu-driven programming
